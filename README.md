# new_dice_app

A new Flutter project.
A simple Flutter app with two interactive dice that roll and change numbers when tapped.

## Getting Started

This project is a starting point for a Flutter application.
This is a simple Flutter application that displays two dice on the screen. 
When the user taps on a dice, both dice roll and update to new random numbers.
The app demonstrates basic Flutter concepts such as gesture handling, state management, and random number generation.
